# Pasta de Música

## Instruções

1. Adicione sua música de fundo aqui
2. Nomeie o arquivo como: `background.mp3`
3. Formatos suportados: MP3, OGG, WAV
4. O arquivo será tocado em loop automaticamente

## Observações

- A música começa a tocar após o usuário clicar no botão "Abrir" na tela inicial
- O volume está configurado para 50% por padrão
- Há um botão de play/pause fixo no canto inferior direito da tela
- Se o arquivo não for encontrado, o site continuará funcionando normalmente (apenas sem música)

## Onde conseguir música?

Certifique-se de usar música sem direitos autorais ou música que você tenha permissão para usar.

Algumas opções:
- YouTube Audio Library (https://www.youtube.com/audiolibrary)
- Free Music Archive
- Música própria ou de domínio público
