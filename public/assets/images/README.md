# Pasta de Imagens

## Instruções

1. Adicione suas fotos aqui (15-25 fotos recomendadas)
2. Nomeie como: `foto1.jpg`, `foto2.jpg`, `foto3.jpg`, etc.
3. Formatos suportados: JPG, PNG, WEBP
4. Resolução recomendada: máximo 1920x1920px

## Depois de adicionar as fotos

Atualize os caminhos e legendas no arquivo:
`/src/data/photos.js`

Exemplo:
```javascript
{
  id: 1,
  src: '/assets/images/foto1.jpg',
  caption: 'nosso primeiro encontro',
  rotation: -3,
}
```
