# Site de Aniversário — Projeto

## Objetivo

Criar um pequeno site interativo de aniversário que conte uma história emocional através de fotos, pequenos momentos e interações.
A experiência termina com uma carta pessoal desbloqueada após um puzzle (sudoku).

O site deve parecer **delicado, artesanal e íntimo**, não tecnológico.

---

# Conceito Geral

## Tema

**“Pequenas coisas que formam a nossa história”**

A experiência deve lembrar:

* um álbum de memórias
* um caderno de anotações
* algo artesanal

Inspirado em:

* papel
* bordado
* pequenos desenhos
* polaroids

---

# Estética Visual

## Estilo

* delicado
* romântico
* simples
* levemente vintage

## Paleta de cores sugerida

* branco papel (#f8f6f2)
* bege claro (#efe9df)
* rosa suave (#e8cfd1)
* verde suave (#cfd8c6)
* cinza texto (#4a4a4a)

## Elementos gráficos

* textura de papel no fundo
* desenhos pequenos de **lírios**
* fotos estilo **polaroid**
* cantos com pequenas ilustrações delicadas

## Animações

Somente animações suaves:

* fade-in
* leve movimento nas fotos
* elementos aparecendo gradualmente

Nada chamativo ou tecnológico.

Música principal do site:

**Cravo e Canela**

* botão para pausar / tocar novamente

# Estrutura do Site

Fluxo narrativo:

1. Tela de abertura
2. Primeira interação (piada interna das bochechas)
3. Galeria de memórias
4. Estatísticas do relacionamento
5. Puzzle (sudoku)
6. Carta
7. Encerramento

---

# 1. Tela de Abertura

Conteúdo central:

Feliz aniversário, [nome dela]

Eu fiz uma pequena surpresa para você.

Botão principal:

"Abrir"

Elementos visuais:

* fundo papel
* um desenho delicado de lírio
* estética minimalista

A música começa após clicar no botão.

---

# 2. Primeira Surpresa (Piada Interna)

Pergunta:

"Antes de continuar…

uma pergunta importante

você ainda gosta das minhas bochechas?"

Botões possíveis:

* sim
* óbvio
* muito

Todos levam para a próxima página.

Objetivo:

* criar humor
* criar conexão imediata

---

# 3. Galeria de Memórias

Layout:

* mural de fotos
* fotos estilo polaroid
* levemente inclinadas

Quantidade sugerida:
15–25 fotos

Cada foto pode ter uma pequena legenda.

Exemplos de legendas:

* nosso primeiro encontro (que terminou às 5 da manhã)
* um dia qualquer que ficou especial
* a viagem do Airbnb
* um momento feliz
* um dia normal que virou lembrança

As fotos podem aparecer gradualmente na tela.

---

# 4. Estatísticas do Nosso Namoro

Seção divertida com pequenos dados sobre o relacionamento.

Título:

"Algumas estatísticas nossas"

Ideias de estatísticas:

Tempo juntos

* desde fevereiro de 2024
* contador automático em dias

Número aproximado de:

* cafés juntos
* bares visitados
* fotos tiradas
* risadas
* horas conversando

Outras ideias:

"Primeiro encontro"

* duração aproximada: 8 horas

"Quantidade de bochechas apertadas"

* incontável

"Primeira viagem"

* Airbnb especial

"Primeira casa juntos"

* recém conquistada

Essa seção pode ser apresentada como **cards ou números grandes**.

---

# 5. Puzzle (Sudoku)

Mensagem inicial:

"Para abrir minha carta
você precisa resolver um pequeno desafio"

Tipo de puzzle:
Sudoku simples.

Requisitos:

* dificuldade média
* resolução possível em poucos minutos
* interface simples

Quando o sudoku for resolvido corretamente:

Mostrar botão:

"Abrir a carta"

---

# 6. Carta

Essa é a parte mais importante do site.

Layout:

* texto central
* fonte elegante
* bastante espaço em branco

Estrutura sugerida da carta:

1. Introdução
   explicar por que fez o site

2. Relembrar o começo
   mencionar como se conheceram

3. O primeiro encontro
   noite longa até 5 da manhã

4. Momentos marcantes
   pequenas memórias

5. Admirar quem ela é
   arquiteta
   criativa
   borda
   gosta de puzzles

6. A nova casa
   morar juntos
   novo capítulo

7. Conclusão
   agradecer por estar com ela

---

# 7. Encerramento

Mensagem final:

"feliz aniversário ❤️"

"obrigado por dividir a vida comigo
e por gostar das minhas bochechas"

Mensagem final pequena:

"ainda temos muitas fotos para tirar juntos"

Opcional:
botão para voltar para o início ou rever fotos.

# Detalhes Técnicos

## Tecnologias sugeridas

HTML
CSS
JavaScript simples

## Componentes necessários

* player de música
* galeria de fotos
* contador de tempo juntos
* sudoku funcional
* animações suaves

## Experiência

O site deve parecer:

* íntimo
* calmo
* emocional
* artesanal

Evitar aparência de:

* app
* produto tecnológico
* interface complexa

Carta

Estatísticas

* estimar números divertidos

Sudoku

* usar um sudoku pronto em JavaScript

# Objetivo Final

Criar uma pequena experiência que faça a pessoa:

1. sorrir
2. lembrar de momentos juntos
3. se emocionar com a carta final
